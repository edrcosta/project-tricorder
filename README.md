# project-tricorder

A modular 8bit handheld computer plataform to study eletronics. 
a crazy dream that slowly become reallity 

## Objective 

Build a modular handhelt 8 bit retro computer "with a power of an gameboy".

## the computer 

### Components 

1. 2x - Atmega328p 
2. 2x - 8051 
3. 1x - z80a
